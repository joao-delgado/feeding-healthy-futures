# High Protein Rush

## Description
“high-protein” was the number one ranked eating pattern from the IFIC 2023 _Food & Health Survey_ (at 18%, followed by mindful eating [17%], calorie-counting [12%], clean eating [12%], and intermittent fasting [12%]).
